# PUFI LANDING PAGE

Test assignment de IT Globers. Página landing de productos PUFI.

## Cómo instalar

1. Clonar repositorio localmente.
2. Instalar dependencias: `npm install`.
3. Iniciar servidor locar: `npm start`.
